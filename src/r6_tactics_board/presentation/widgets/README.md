# Widgets

可复用 Qt 部件按场景域拆分。这里的代码应尽量是“可被页面组装的控件”，而不是页面流程本身。

## 子目录职责

- `canvas/`: 地图画布、场景和图元。
- `editor/`: 编辑页专用但可独立维护的面板和输入组件。
- `timeline/`: 时间轴控件。

## 适合放这里的内容

- 单独可实例化的 Widget。
- 画布视图、场景、图元和相关交互。
- 面板、工具栏、时间轴表格等 UI 组件。

## 不要放这里的内容

- 页面导航和主窗口编排。
- 关键帧纯数据规则。
- 资源扫描和工程持久化。

## 落点规则

- 只要一个控件可能被两个页面复用，就优先放这里。
- 如果一个控件已经需要知道页面的全部状态，说明它还没抽干净，先留在页面目录更合适。
