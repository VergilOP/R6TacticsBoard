# Esports Page

这里放电竞历史数据浏览页。

## 当前文件

- `esports_page.py`: 地图维度的电竞历史记录浏览与详情展示页。

## 适合放这里的内容

- 地图切换、比赛列表、字段概览、原始 JSON 预览。
- 页内展示用的小型本地 Widget。
- 只服务电竞历史浏览页的交互反馈。

## 不要放这里的内容

- 电竞数据目录扫描和 JSON 解析实现。
- 干员资源索引底层逻辑。
- 其他页面也要复用的通用控件。

## 落点规则

- 如果只是电竞页自己的展示逻辑，继续放这里。
- 如果后面要做多个电竞相关页面，共享的数据读取能力优先放到 `infrastructure/esports/`。
- 页面默认优先展示“观赛信息”，字段排查和原始 JSON 放到次级数据视图，不反向污染主视图。
