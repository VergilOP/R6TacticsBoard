# Routing

这里放与地图互动点、楼层切换、路径分段和插值相关的应用逻辑。

## 适合放这里的内容

- 互动点图之间的寻路。
- 手动互动点链路的标准化和补全。
- 播放路径分段、插值、跨楼层跳转规则。

## 不要放这里的内容

- `QGraphicsScene` 中图元如何绘制。
- 互动点按钮、表单、提示文字。
- 工程文件如何保存这些路径结果。

## 当前文件

- `interaction_routing.py`: 跨楼层互动点寻路、手动互动点前缀、播放路径分段与插值。

## 新文件落点

- 新增互动点算法时，先扩这里，再决定页面是否需要新的输入控件。
- 如果只是图元显示样式变化，去 `presentation/widgets/canvas/`，不要把 UI 状态塞回来。
