# Timeline

这里放时间轴相关的纯数据编辑逻辑，不负责 Qt 表格本身如何渲染。

## 适合放这里的内容

- 关键帧列的插入、删除、移动。
- 单元格写入、清空、复制等数据变更。
- 行顺序调整、时间轴结构重建。

## 不要放这里的内容

- 表头样式、快捷键注册、鼠标事件分发。
- 撤销重做历史本体。
- 播放进度条和定时器更新。

## 当前文件

- `timeline_editor.py`: 关键帧列、单元格、行顺序等纯数据变更控制器。

## 新文件落点

- 只要逻辑输入输出可以写成“传入数据，返回新数据或变更结果”，优先放这里。
- 如果逻辑已经开始读取 `QTableWidgetItem` 或选择状态，说明它更接近 `presentation/widgets/timeline/`。
