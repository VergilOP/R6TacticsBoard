# Playback

这里放播放流程相关对象，负责播放状态推进，而不是具体播放控件外观。

## 适合放这里的内容

- 播放状态、暂停、恢复、倍速、步进。
- 时间推进和播放索引切换规则。
- 与页面无关的播放控制器接口。

## 不要放这里的内容

- 播放浮层的按钮布局。
- 时间轴表格选中高亮。
- 工程文件读写和资源定位。

## 当前文件

- `controller.py`: 基础播放状态与控制器。

## 新文件落点

- 如果新功能是“播放应该如何推进”，放这里。
- 如果新功能是“播放栏上要显示什么控件”，放 `presentation/widgets/editor/`。
