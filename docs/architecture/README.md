# Architecture Docs

这里放架构和设计说明文档。

## 当前文件

- `overview.md`: 当前项目的分层结构、核心对象、数据流和设计风险。
- `overview_2_5d_plan.md`: 2.5D 全楼层总览的交互目标、相机模型、当前实现状态和分阶段落地方案。
- `roadmap.md`: `v0.6.x` 之后的功能和结构推进路线。

## 适合放这里的内容

- 分层边界和依赖方向。
- 核心编辑流程和数据流。
- 设计取舍、风险点、后续重构方向。

## 不要放这里的内容

- 资源目录命名细节。
- 打包命令和发布步骤。
- 只服务某个临时调试问题的零散记录。
